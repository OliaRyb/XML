1. Создать внешний репозиторий c названием xml.
2. Клонировать репозиторий xml на локальный компьютер. 				2. git clone git@github.com:OliaRyb/xml.git
											cd xml
3. Внутри локального xml создать файл “new.xml”.				3. touch new.xml
4. Добавить файл под гит.							4. git add new.xml
5. Закоммитить файл.								5. git commit -m "Initial commit"
6. Отправить файл на внешний GitHub репозиторий.				6. git push
7. Отредактировать содержание файла new.xml - написать 				7. cat > new.xml
 информацию о себе (ФИО, возраст, количество домашних животных, 	 
 будущая желаемая зарплата). Всё написать в формате xml.				
8. Отправить изменения на внешний репозиторий.					8. git commit	-m "Added personal data" -a 
9. Создать файл preferences.xml							9. touch preferences.xml
10. В файл preferences.xml добавить информацию о своих 				10. cat > preferences.xml
 предпочтениях (Любимый фильм, любимый сериал, любимая еда, 		
 любимое время года, сторона которую хотели бы посетить) в 			 
 формате xml.																
11. Создать файл sklls.xml добавить информацию о скиллах, 			11. touch skills.xml
 которые будут изучены на курсе в формате xml.						cat > skills.xml									
12. Отправить сразу 2 файла на внешний репозиторий.				12. git add ./
												git commit -m "Added new files with data"
												git push
13. На веб интерфейсе создать файл bug_report.xml.
14. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
15. На веб интерфейсе модифицировать файл bug_report.xml, 		
 добавить баг репорт в формате xml.
16. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
17. Синхронизировать внешний и локальный репозиторий xml.			17. git pull
